# ☀️ Morning Digest — Saturday, March 15

## 🤖 NVIDIA Drops Nemotron 3 Super for Autonomous AI Agent Swarms

NVIDIA released Nemotron 3 Super, a 120-billion parameter open-weight model specifically designed for complex agentic workflows. The model combines advanced reasoning with hybrid mixture-of-experts architecture, targeting AI agents that need to collaborate and maintain context across multi-step tasks.

**Key Features:**
- 120B parameters optimized for agent coordination
- Open-weight availability on Hugging Face, OpenRouter, Perplexity
- Hybrid mixture-of-experts for efficient inference
- Large context windows for multi-step workflows

🔗 [Full Story](https://www.indiatvnews.com/technology/news/nvidia-nemotron-3-super-open-source-ai-model-launched-for-agentic-systems-2026-03-13-1033651)
